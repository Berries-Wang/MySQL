# Next Key Lock
## 简要
&nbsp;&nbsp;A next-key lock is a combination of a record lock on the index record and a gap lock on the gap before the index record.
- Next Key锁是由 位于索引记录上的记录锁 和 位于索引记录之前间隙的间隙锁 组成
