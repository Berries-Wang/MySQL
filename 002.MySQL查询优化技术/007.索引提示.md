# 索引提示
&nbsp;&nbsp;MySQL数据库支持索引提示(INDEX HINT),显式地告诉优化器使用哪个索引。
- USE INDEX
   + ```sql
       # USE INDEX 只是告诉优化器可以选择该索引，实际上优化器还是会根据自己的判断进行选择
       SELECT * FROM T USE INDEX(a) WHERE A = 1 AND B=2;
      ```
- FORCE INDEX
   + ```sql
         # 强制使用指定索引
         SELECT * FROM T FORCE INDEX(a) WHERE A = 1 AND B=2;
      ```
